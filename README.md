# Customer Segmentation using K-Means Clustering

## Project Overview

This project uses K-Means Clustering to segment customers into distinct groups based on their annual income and spending behavior. Customer segmentation helps businesses understand purchasing patterns, improve marketing strategies, and enhance customer retention.

## Dataset

* Dataset: Mall Customers Dataset
* Records: 200 customers
* Features:

  * CustomerID
  * Gender
  * Age
  * Annual Income (k$)
  * Spending Score (1-100)

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn
* Google Colab

## Methodology

1. Data Loading and Exploration
2. Feature Selection
3. Elbow Method for Optimal Cluster Selection
4. K-Means Clustering
5. Cluster Visualization
6. Business Insights and Recommendations

## Customer Segments Identified

| Segment                   | Annual Income | Spending Score |
| ------------------------- | ------------- | -------------- |
| High Value Customers      | 86.54         | 82.13          |
| High Income, Low Spending | 88.20         | 17.11          |
| Low Income, High Spending | 25.73         | 79.36          |
| Low Value Customers       | 26.30         | 20.91          |
| Average Customers         | 55.30         | 49.52          |

## Key Findings

* Customers were successfully segmented into five distinct groups.
* High Value Customers represent the most profitable segment.
* High Income, Low Spending customers present opportunities for targeted promotions.
* Customer segmentation helps improve marketing effectiveness and customer engagement.
* K-Means clustering provides actionable business insights from customer behavior data.

## Business Recommendations

* Offer loyalty rewards to High Value Customers.
* Target High Income, Low Spending customers with personalized campaigns.
* Improve customer retention strategies using segmentation insights.
* Optimize marketing budgets based on customer segment value.
